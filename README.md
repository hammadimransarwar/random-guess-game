# 🎯 Random Guess Game

Welcome to the **Random Guess Game**! This is a clean, lightweight, and fully responsive web game built from scratch using vanilla web technologies. 

The concept is simple: the computer picks a secret number between 1 and 10, and it's your job to figure it out. To help you out, the game drops real-time hints telling you if your guess is too high or too low.

---

## ✨ Features

* **Fresh Numbers Every Time:** Generates a brand-new secret number automatically the second the page loads.
* **Bulletproof Validation:** You can't break the game! The input filter instantly catches accidental blank submissions, letters, negative numbers, or anything higher than 10.
* **Smart Feedback:** Real-time hints steer you in the right direction (Too high! 📈 or Too low! 📉).
* **Play Anywhere:** Styled with mobile-first CSS media queries, meaning it looks and plays great whether you are on a desktop monitor or tapping away on your phone.

---

## 🛠️ Built With

* **HTML5** – For semantic, accessible form structures.
* **CSS3** – For custom layouts, vibrant colors, and responsive `@media` breakpoints.
* **JavaScript (ES6)** – For DOM manipulation, math logic, and form event handling.
